Test the Search Engine

Before we test the search engine, we need to write the PageRank output to the HBase clueWeb09PageRankTable.

# upload the pagerank result to hbase “clueWeb09PageRankTable”

hadoop jar /root/software/hadoop-1.1.2/lib/cglHBaseMooc.jar  iu.pti.hbaseapp.clueweb09.PageRankTableLoader  /root/hbaseMoocAntProject/resources/en0000-01and02.docToNodeIdx.txt  /root/hbaseMoocAntProject/resources/en0000-01and02_reset_idx_and_square_pagerank.out

Now, combined with "Building an Inverted Index" , we have built three database tables on HBase:

  • clueWeb09DataTable
  • clueWeb09IndexTable
  • clueWeb09PageRankTable

Using the provided test program, the following commands search the keyword “database”, and the dataflow is shown as Figure 1.
