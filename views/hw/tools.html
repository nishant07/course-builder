Goal

The Goal of this Lab is to prepare the environment required for all the homework assignments. Also this lab describes the homework environment so that the student can get familiar with it.

Deliverables

None

Introduction

This course has homework ranging from configuration of the environment and various tools to writing advanced programs. The homework is packaged in a virtual machine to allow students focus on the specific tasks rather than focusing on things like environment setup which can be complicated in distributed systems.

As mentioned above the homework is packaged in a virtual machine and the student is expected to download this virtual machine and install it on their home computers.

Installing the Virtual Machine

First thing is to install VirtualBox to run the virtual machine.

Download the latest version of the VirtualBox and install it on your local machine: VirtualBox downloads.

After downloading the VirtualBox and installing it you need to download the Virtual Machine Image

Download the prepackaged appliance from this link.

Running the Virtual Machine

Start the VirtualBox and go to menu File -> Import Appliance

Select the downloaded Virtual Machine image and import it to the VirtualBox

After the import is successful you can start the virtual machine

File structure

All the files related to homework are on the /root directory of the machine

Here is a description of the files in the homework

MoocHomeworks This folder contains all the homework material
data Contains data required by the homework
software Contain the software required by the homework

Different homework will give a detailed description about the files required

Development Environment

The students are expected to complete the homework using the Java programming language.

We have installed Eclipse IDE in the Virtual machine for the development purposes. Students can use the Eclipse IDE or use another editor like VIM, gEdit for editing the programs.

Compiling & Running the Homework

The homework uses Ant build system to compile and package the homework. Most of the homework have a build.sh file that will assist the student to compile the program.
