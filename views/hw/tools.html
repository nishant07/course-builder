Goal

The goal of this lab is to prepare the environment required for all subsequent homework assignments. This will also permit you to become familiar with the basic homework setup.

Deliverables

None

Introduction

This course has homework covering a wide range of topics, from configuration of the environment and various tools to writing advanced programs. It all comes packaged in a virtual machine to allow students to focus on specific tasks instead of environment setup, which can be complicated in distributed systems. You are expected to download the provided virtual machine and install it on your own computer.

Installing the Virtual Machine

First off you will need VirtualBox to run the virtual machine. Download the latest version of the VirtualBox and install it on your local machine: VirtualBox downloads.

Afterwards you need to download the Virtual Machine Image from this link.

Running the Virtual Machine

Start the VirtualBox and go to menu File -> Import Appliance. Select the downloaded Virtual Machine image and import it to the VirtualBox. After the import is successful you can start the virtual machine.

File structure

All the files related to homework can be found on the /root directory of the machine:

MoocHomework This folder contains all the homework material
data Contains data required by the homework
software Contains the software required by the homework

Later homework will give a detailed description about the files required.

Development Environment

Students are expected to complete the homework using Java programming language. We have installed Eclipse IDE in the Virtual machine for the purpose of development. You can either use this or another product like VIM and gEdit for editing the programs.

Compiling & Running the Homework

The homework uses Ant build system to compile and package itself. Most have a build.sh file that will assist you in compiling the program.
